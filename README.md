# Поймай AI

Бесплатный детектор машинного текста для русского языка. Работает прямо в браузере — текст никуда не уходит, всё считается у вас на устройстве.

Это «линтер примет»: он показывает признаки, по которым текст похож на написанный нейросетью — клише, ровный ритм, невидимые символы, рубленые фразы и другие. Это подсказка, а не приговор: грамотный человек тоже может набрать высокий процент, а короткий вылизанный машинный текст может проскочить.

## Файлы

- `index.html` — сама страница
- `banlist.js` — список фраз-примет (можно свободно дополнять, не трогая страницу)
- `m6.png` — иллюстрация

Ни сборки, ни сервера не нужно. Можно просто открыть `index.html` в браузере или включить GitHub Pages.
